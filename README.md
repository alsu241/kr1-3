2. Для начала работы откройте форму входа под названием "ФормаВхода".
3. Для доступа под ролью исследователя используйте следующие данные:
      Логин: ivanov;
      Пароль: ivanov123
      
      Для доступа под ролью администратора используйте следующие данные:
      Логин: admin;
